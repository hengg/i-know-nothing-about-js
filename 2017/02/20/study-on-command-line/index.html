什么是命令行

命令行是用户通过键盘输入字符指令、计算机输出字符结果的一种人机交互的方式。在熟记命令的前提下，使用命令行往往要较使用图形用户界面的操作速度要快，并且更容易执行批量操作。

Bash是比较常用的命令行shell，它是大多数Linux系统以及Mac OS X默认的shell，它能运行于大多数类Unix风格的操作系统之上，甚至被移植到了Microsoft Windows上的Cygwin系统中，以实现Windows的POSIX虚拟接口。

Cygwin是一个在windows平台上运行的类UNIX模拟环境。

shell是指操作系统最外层（壳），通常指的是命令行界面的解析器。

POSIX表示可移植操作系统接口（Portable Operating System Interface），POSIX标准定义了操作系统应该为应用程序提供的接口标准，是IEEE为要在各种操作系统上运行的软件而定义的一系列API标准的总称。

高效/批量

强力工具

有许多开发中常用的强力工具和命令行更搭，比如Vim，Git和npm

Vim是一个功能强大、在程序员中被广泛使用的文本编辑器。

Git是Linus Torvalds开发的分布式版本控制工具,几乎所有的开源软件都在使用Git，同时使用了Git的github也是世界上最受欢迎的软件项目托管平台

NPM (Node Package Manager)，是一个NodeJS包管理和分发工具，已经成为了事实上发布Node模块的标准。
前端常用模块如SASS，LESS，webpack，babel等都需要使用npm命令来安装。

命令行格式

常用操作

  • 目录操作

    cd：Change Directory的缩写，切换目录

    pwd：Print Working Directory的缩写，显示当前目录

    mkdir [-p] directoryname：mkdir是Make Directory的缩写，创建目录; 使用参数-p时若路径中的某些目录尚不存在，将创建这些尚不存在的目录。

  • 查看路径

    ls [-al] path：ln是List的缩写，查看路径；参数a：列出目录下的一切文件，包括隐藏文件；参数l：列出目录下文件的具体信息，包括权限，大小，创建者等等

  • 创建文件

    cho '1' > filepath：创建文件

    touch filename：创建文件，如果文件已存在则改变文件更新时间

  • 复制及移动

    cp [-r] source_file target_file：cp是cpoy的缩写，复制文件；参数r代表recursive，递归复制目录及其内容

    mv source target：mv是move缩写，移动节点

  • 删除

    rm [-fr] file：rm是remove的缩写，删除文件；参数f代表force，忽略提示；参数r代表recursive递归删除目录及其内容

  • 其他

    tree：查看目录结构

    ln source_file target_dir：建立软链接 （ Windows 系统默认不支持链接）
