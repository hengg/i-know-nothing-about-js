什么是JavaScript

JavaScript是一门脚本语言（脚本语言不具备开发操作系统的能力），它本身不提供I/O相关的API，需要宿主环境（host）提供。最常见的环境就是浏览器，JavaScript最初的目的就是内置于浏览器用于DOM操作；它也有服务器环境，大名鼎鼎的Node。

目前JavaScript最新版本简称为ES6。

什么是ES6？
ES6泛指EMCAScript 5.1版以后的JavaScript的下一代标准，涵盖了ES2015、ES2016、ES2017等等

什么是ECMAScript？
ECMA是欧洲计算机制造商协会的简称，1997年为解决早期JavaScript不同版本并存的问题，JavaScript被提交给EMCA以制定该脚本语言的标准，这个标准称为EMCAScript。各大浏览器厂商均支持这个标准。

声明前置：JavaScript引擎的工作方式是，先解析代码，获取所有被var声明的变量，然后再一行一行地运行。这造成的结果，就是所有的变量的声明语句，都会被提升到代码的头部，这就叫做声明前置（hoisting）。但使用let和const关键字则不会声明前置。

在ES6下作用域有三种，全局作用域、函数作用域和块作用域。

  • 全局作用域：该变量在代码中任何地方都能访问到，最外层函数和在最外层函数外面定义的变量拥有全局作用域，所有末定义直接赋值的变量自动声明为拥有全局作用域；
  • 函数作用域：该变量在函数内部声明，只能在函数内部访问；
  • 块作用域：该变量在块内声明，只能在代码块内访问（块可以理解为花括号内）；
