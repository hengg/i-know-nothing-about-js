HTML中的常用meta

发表于 2017-03-08 |

什么是meta？

在了解字符编码的过程中我们曾在html的head标签中见过<meta charset="utf-8">，在这里通过meta来声明页面的字符编码格式为utf-8。今天就进一步了解meta

MDN中对于meta的定义如下：元素表示那些不能由其它HTML元相关元素 (<base>, <link>, <script>, <style> 或 <title>) 之一表示的任何元数据信息。

meta有什么用？

浏览器不会渲染meta元素，但仍然会解析它。此外搜索引擎也可以读取meta元素。因此我们可以通过meta向浏览器和搜索引擎传递信息。

怎样使用meta？

meta元素通常在head元素中声明。meta元素有四个重要属性，分别是name，http-equiv，以及为以上两种属性提供相关值的content和我们熟悉的charset。

charset

声明当前文档所使用的字符编码，虽然规范中没有对字符编码做要求，但推荐使用utf-8。

name

name和content成对出现，把content值关联到一个name属性上，举例来说：<meta name="author" content="hengg">即声明文件作者为hengg。

常用name属性有如下几种：

viewport

优化页面的显示，常用于响应式开发。

author，description和keywords

搜索引擎通过以上meta声明能够得知该页面的作者、关键字和主要内容。

robots
